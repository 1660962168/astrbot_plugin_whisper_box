# 碎碎念收集器 (WhisperBox) 

> “你说，我在听。” —— 让你的 AI 机器人拥有真人的耐心与温度。

这是一个为 [AstrBot](https://github.com/Soulter/AstrBot) 框架编写的陪伴型增强插件。

在日常聊天中，人类的发言往往是碎片化、想到哪说到哪的。但传统的 AI 往往在收到第一句话时就急着回复，导致用户必须把话憋成一篇“小作文”才能发送。
**碎碎念收集器** 改变了这一现状。它会像一个耐心的倾听者，默默收集你连续发送的短句，直到你停顿（自定义秒数）后，再将这些话语整理合并，交给 AI 进行一次完整、贴心的回复。

## ✨ 核心特性

* ⏳ **智能延迟收集**：自定义等待时间（如 5 秒），只要用户在倒计时内继续输入，计时器自动重置。
* 🎛️ **原生 Web UI 配置**：完美接入 AstrBot 的可视化控制台，支持网页端直接修改参数和生效范围（全部/仅私聊/仅群聊）。
* ⚡ **指令直通车**：遇到带有 `/` 前缀的指令（如 `/画图`、`/天气`），立刻放行并结算当前收集池，绝不耽误正常功能。
* 🛡️ **防乱序与防串线**：
  * 基于底层物理 **时间戳 (Timestamp)** 强制排序，彻底粉碎高并发导致的消息乱序问题。
  * 基于 `群聊ID + 发送者ID` 的双重隔离，群聊再活跃也绝不串线。

## 📦 安装说明

1. 确保你的机器人正在使用 **AstrBot** 框架。
2. 在 AstrBot 控制台的插件市场中安装（如已上架），或者通过 在 AstrBot WebUI 中进入“插件管理”，选择从 Git 安装，填写仓库地址：：
   ```bash
   git clone https://github.com/1660962168/astrbot_plugin_whisper_box
   ```
3. 在 AstrBot 控制台重启机器人或重载插件即可生效。

## 🛠️ 使用与配置

本插件支持两种配置方式：**聊天框指令** 和 **插件配置**。数据双向同步，实时生效。

### 聊天框指令

* **指令**：`/等待 [秒数]`
* **说明**：快速设置接收消息的等待时长。
* **示例**：发送 `/等待 8`，机器人将会在每次你发完消息后等待 8 秒，确认你没话说了再进行回复。

### 网页控制台配置

进入 AstrBot 的 Web 管理面板，找到 `碎碎念收集器` 插件的设置页面，你可以调整以下参数：
1. **等待时间（秒）**：支持小数（如 5.0）。
2. **生效范围**：下拉选择 `全部`、`仅私聊` 或 `仅群聊`。

## 🐛 常见问题

**Q: 为什么我发了很长一段话，AI 没有马上回复我？**
A: 因为插件正在“耐心等待”你是否还有补充哦！等待时间结束后，AI 就会回复你。如果你觉得等太久，可以通过 `/等待` 指令把时间调短一点。

**Q: 我在等待期间输入了其他机器人的 `/指令`，会被拦截吗？**
A: 不会！插件内置了智能识别，一旦检测到你发送的是指令，会立刻放行，同时会把之前已经收集到的“碎碎念”瞬间提交给 AI，丝滑无卡顿。
